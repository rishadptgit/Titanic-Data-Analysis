# Titanic-Data-Analysis
Analyzed 891 passenger records from Kaggle Titanic dataset using Python/Pandas/Seaborn. Cleaned 20% missing Age data, created 15+ visualizations. Key findings: Females survived 74% vs males 19% (3.5x); 1st class 63% vs 3rd class 25%. Production-ready notebook with cleaned CSV. Makes  data cleaning, EDA, visualization skills for  analyst roles.
